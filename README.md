# Airbnb Market Analysis Dashboard

## Project Overview
This Tableau dashboard analyzes Airbnb listing data to identify pricing trends, annual revenue opportunities, and high-performing zip codes for short-term rental investments.

## Business Questions
- How does bedroom count affect average nightly price?
- Which zip codes command the highest prices?
- Which neighborhoods generate the most revenue?
- How does revenue trend throughout the year?

## Dataset
Seattle Airbnb Open Data containing listing prices, bedroom counts, zip codes, and calendar information.

## Tools Used
- Tableau
- Data Visualization
- Calculated Fields
- Dashboard Design
- Business Analysis

## Key Insights
- Larger properties generated higher nightly rates.
- Certain zip codes consistently outperformed others.
- Revenue increased steadily throughout the year.
- One-bedroom listings represented the majority of inventory.

## Business Recommendation
Focus on high-performing zip codes and larger properties to maximize revenue potential while monitoring occupancy trends and acquisition costs.
